Hack to Learn

Light bulb!

The Hackatron project is an ethical web hacker simulation for the students of West Visayas State University - Institute of Information and Communications Technology. It aims to help students in sharpening their code analysis skills and solve programming problems even if they are facing unfamiliar codes.

 

Advancing to the next level of the game require different web development skills (e.g. Networking, HTML 5, CSS 3, JavaScript, PHP(soon) and MySQL(soon).

Disclaimer

This project is not endorsing the unethical use of hacking or cracking. This is only for educational purposes.
